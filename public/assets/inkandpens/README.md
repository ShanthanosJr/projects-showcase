# Ink & Pen Bookstore - Assets

This directory contains the media assets for the Ink & Pen Bookstore project.

## Project Overview
A boutique-style bookstore e-commerce application with a vintage-inspired design, focused on book discovery, browsing, and catalog management through a clean and intuitive user experience.

## Assets Included
- **Screenshots:** 14 UI screenshots (inp1.png to inp14.png)
- **Demo Video:** inp_v.mp4 (screen recording of the application)

## Technology Stack
- **Frontend:** React
- **Backend:** ASP.NET Core
- **Database:** SQL
- **Deployment:** Responsive web application

## Key Features Demonstrated
- Advanced book listing with category-based sections
- Search, filtering, and sorting for improved discovery
- Responsive vintage-themed UI inspired by classic bookstores
- FAQ, ratings, and reviews for customer trust
- Clean REST API integration with a relational database
